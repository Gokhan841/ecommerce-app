# E-Ticaret Uygulaması

Bu proje, modern web teknolojileri kullanılarak geliştirilmiş bir e-ticaret platformudur.

## Kullanılan Teknolojiler

- **React.js** – Kullanıcı arayüzü
- **React Router DOM** – Sayfa yönlendirmeleri
- **Axios** – API istekleri
- **JWT (JSON Web Tokens)** – Kimlik doğrulama
- **RESTful API** – Backend ile iletişim
- **Formik** – Form yönetimi
- **Yup** – Form validasyonu
- **Tailwind CSS** – Stil ve responsive tasarım
